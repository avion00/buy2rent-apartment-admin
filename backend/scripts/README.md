# Scripts Directory

This directory contains utility scripts organized by purpose.

## Directory Structure

- **setup/** - Installation and environment setup scripts
- **testing/** - Test scripts for various features and APIs
- **debugging/** - Diagnostic and debugging utilities
- **migration/** - Database migration and data migration scripts
- **data_management/** - Data manipulation, import/export, and cleanup utilities

## Usage

Most scripts should be run from the backend root directory:

```bash
cd /root/buy2rent/backend
python scripts/<category>/<script_name>.py
```

## Notes

- Ensure your virtual environment is activated before running scripts
- Some scripts may require specific environment variables to be set
- Check individual script documentation for specific requirements
